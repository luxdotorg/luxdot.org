# LuxDot Homepage

GitHub Pages-ready single-file homepage.

## Deploy
1. Upload `index.html` to the repository root.
2. Open **Settings → Pages**.
3. Choose **Deploy from a branch**.
4. Select the main branch and `/ (root)`.

## Audio
Modern browsers block autoplay audio. The page uses a deliberate **ENTER SIGNAL** button.
After that single click, ambient noise, hums, tones, heartbeat and spoken signal words start.
